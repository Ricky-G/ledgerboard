# Support

Use [GitHub Issues](https://github.com/Ricky-G/kanban-ledger/issues) for reproducible bugs and focused
feature requests. Use [GitHub Discussions](https://github.com/Ricky-G/kanban-ledger/discussions) for
workflow ideas, examples, and general questions.

Before posting:

1. Run **Kanban Ledger: Validate Board Bundle**.
2. Check the latest release notes and existing issues.
3. Remove private entity names, descriptions, and history before sharing files.

Security concerns belong in a private security advisory, as described in `SECURITY.md`.
